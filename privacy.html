隐私政策

我们重视并保护您的隐私

最后更新时间：2025年

隐私政策概述

感谢您使用文本转语音在线工具。我们深知个人隐私的重要性，并致力于保护您的隐私安全。本隐私政策旨在帮助您了解我们如何处理您的信息。

信息收集与使用

文本数据：

  • 输入文本：您输入的文本内容将通过 WebSocket 连接发送至微软语音合成服务进行语音转换。文本数据仅用于实时合成，不会被本工具存储或记录。
  • 本地处理：当使用浏览器内置语音引擎（备用方案）时，文本数据完全在您的浏览器本地处理，不会发送到任何服务器。

技术信息：

  • 我们可能收集浏览器类型、操作系统等基本技术信息，用于改进服务兼容性。
  • 本工具不要求用户注册或登录，不收集任何个人身份信息。

第三方服务

本工具使用微软 Edge TTS 服务进行语音合成。当您使用本工具时：

  • 您输入的文本内容将以 SSML 格式通过加密的 WebSocket 连接（WSS）发送至微软语音服务。
  • 微软将处理文本并返回合成的音频数据。
  • 微软可能根据其隐私政策收集和处理相关数据。
  • 我们建议您查阅 微软隐私声明 了解详情。

数据安全

我们采取以下措施保护您的数据安全：

  • 与微软语音服务的通信使用加密的 WebSocket（WSS）连接。
  • 本工具不设后端服务器，不存储任何用户文本或音频数据。
  • 合成的音频数据仅保存在您的浏览器内存中，关闭页面后自动清除。
  • 下载的 MP3 文件保存在您的本地设备上，由您自行管理。

本地存储

关于浏览器本地存储：

  • 本工具不使用 Cookies。
  • 本工具不使用 localStorage 或 sessionStorage 存储用户数据。
  • 所有设置（语言、语音角色、语速、音调）仅在当前会话中保持，刷新页面后恢复默认。

用户权利

您拥有以下权利：

  • 随时停止使用本工具。
  • 自行决定输入的文本内容，避免输入敏感的个人信息。
  • 删除已下载的音频文件。
  • 要求了解我们如何处理信息。

隐私政策更新

我们可能会不时更新本隐私政策。重大变更时，我们会在网站上发布通知。建议您定期查看本页面以了解最新的隐私保护措施。

联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • 通过 GitHub Issues 反馈
  • 或访问我们的 关于页面 获取更多信息

重要提示：使用本工具即表示您同意本隐私政策。如果您不同意本政策的任何内容，请停止使用本工具。请注意，您输入的文本内容将发送至微软服务器进行语音合成处理。
