使用说明

  • 在文本框中输入要转换的文字内容（支持中文、英文、日文、韩文等）
  • 选择目标语言和喜欢的语音角色
  • 调节语速和音调参数（可选）
  • 点击"开始转换"按钮生成语音
  • 转换完成后可在线播放或下载 MP3 文件
提示：请使用 python server.py 启动本地服务（需先安装依赖：pip install edge-tts）。如服务端不可用，将自动切换到浏览器内置语音引擎。
