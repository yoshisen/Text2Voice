Privacy Policy

We value and protect your privacy

Last updated: March 2026

Overview

Thank you for using Online Text-to-Speech Tool. We understand the importance of personal privacy and are committed to protecting it. This privacy policy explains how we handle your information.

Information Collection and Use

Text Data

  • Input text: The text you enter is sent through a WebSocket connection to Microsoft's speech synthesis service for conversion. It is used only for real-time synthesis and is not stored or logged by this tool.
  • Local processing: When the browser's built-in speech engine is used as a fallback, text is processed entirely within your browser and is not sent to any server.

Technical Information

  • We may collect basic technical information such as browser type and operating system to improve service compatibility.
  • This tool does not require registration or login and does not collect personally identifiable information.

Third-Party Services

This tool uses Microsoft Edge TTS for speech synthesis. When you use this tool:

  • Your input text is sent in SSML format over an encrypted WebSocket connection (WSS) to Microsoft's speech service.
  • Microsoft processes the text and returns synthesized audio data.
  • Microsoft may collect and process related data according to its own privacy policy.
  • We recommend reviewing the Microsoft Privacy Statement for more details.

Data Security

We take the following measures to protect your data:

  • Communication with Microsoft's speech service uses encrypted WebSocket (WSS) connections.
  • This tool does not store user text or audio data.
  • Synthesized audio data exists only in your browser memory and is cleared when the page is closed.
  • Downloaded MP3 files are stored on your local device and managed by you.

Local Storage

Regarding browser-side storage:

  • This tool does not use cookies.
  • This tool does not use localStorage or sessionStorage to store user data.
  • All settings such as language, voice, speaking rate, and pitch only persist during the current session and return to defaults after a refresh.

User Rights

You have the following rights:

  • Stop using this tool at any time.
  • Choose what text to input and avoid entering sensitive personal information.
  • Delete downloaded audio files.
  • Ask how we handle information.

Policy Updates

We may update this privacy policy from time to time. When major changes occur, we will post a notice on the website. We recommend checking this page periodically for the latest privacy information.

Contact Us

If you have any questions or suggestions about this privacy policy, please contact us through the following methods:

  • Submit feedback through GitHub Issues
  • Or visit our About page for more information

Important: By using this tool, you agree to this privacy policy. If you do not agree with any part of this policy, please stop using the tool. Please note that the text you enter will be sent to Microsoft's servers for speech synthesis.
