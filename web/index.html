常见问题

支持哪些语言和语音角色？

本工具基于 Microsoft Edge TTS 技术，支持中文（普通话、粤语）、英语、日语、韩语、法语、德语、西班牙语等数十种语言，每种语言提供多个语音角色可供选择，包括男声和女声。

转换后的音频是什么格式？

转换后的音频为 MP3 格式，可直接在线播放或下载保存到本地设备，兼容所有主流播放器和设备。

文本长度有限制吗？

单次转换支持最多 1000 个字符。如果您的文本较长，建议分段进行转换。

为什么语音听起来不自然？

请尝试以下方法改善效果：选择与文本语言匹配的语音角色；适当调整语速和音调参数；确保文本中的标点符号使用正确，这会影响语音的停顿和语气。

浏览器内置语音和 Edge TTS 有什么区别？

Edge TTS 是微软提供的云端语音合成服务，音质更自然、语音角色更丰富，需要启动本地服务端。浏览器内置语音（Web Speech API）无需服务端，但音质和可用语音取决于您的操作系统和浏览器，效果可能有所差异。

这个工具免费吗？数据是否安全？

本工具完全免费使用。文本数据仅用于语音合成，不会被存储或用于其他用途。使用本地服务端模式时，所有处理都在您的设备上完成。
